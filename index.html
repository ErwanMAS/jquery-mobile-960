960 grid on jquery mobile

jquery-mobile-960 is a port of 960 grid to jquery mobile, to make it more flexible on tablets. It has the flexibility of 960.gs, and is as easy to use as jquery mobile. well known grid, associate 2 distincts works, 960.gs and jquery mobile Current jquery mobile grid currently are crude at best, not suitable for large screen like tablets.
