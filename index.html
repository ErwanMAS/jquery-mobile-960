960 grid on jQuery mobile

jquery-mobile-960 is a port of 960 grid to jquery mobile. It brings more flexibility to jquery-mobile layout and thus make it more suitable for tablets. It has the flexibility of 960.gs, and is as easy to use as jquery mobile. well known grid, associate 2 distincts works, 960.gs and jquery mobile Current jquery mobile grid currently are crude at best, not suitable for large screen like tablets.

How to use it ?

To get jquery mobile, usually you do something like this: 
    <link href="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.css" />
    <script src="http://code.jquery.com/jquery-1.5.min.js"></script>
    <script src="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.js"></script>
To use 960 grids, simply add one line: 
    <link href="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.css" />
    <link href="css/jquery-mobile-fluid960.css" />
    <script src="http://code.jquery.com/jquery-1.5.min.js"></script>
    <script src="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.js"></script>
