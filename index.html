960 grid on jQuery mobile

jquery-mobile-960 is a port of 960 grid to jquery mobile. It has the flexibility of 960.gs, and is as easy to use as jquery mobile. It brings more flexibility to jquery-mobile layout and thus make it more suitable for tablets.

jqQuery mobile grid is currently rather crude ... uselessly difficult to use on large screens, like tablets. They only split the width in even parts, providing little controls to the designer. It uses an custom API with names ending with a, b, c or d, ui-block-a for example. Not the classic grid-4 or span-3. See more

jquery-mobile-960 aims to fix those 2 issues by brinding more flexibility and known API.

How to use it ?

To get jquery mobile, usually you do something like this: 
    <link href="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.css" />
    <script src="http://code.jquery.com/jquery-1.5.min.js"></script>
    <script src="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.js"></script>
To use 960 grids, simply add one line: 
    <link href="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.css" />
    <link href="http://jeromeetienne.gihub.com/jquery-mobile-960/css/jquery-mobile-960.min.css" />
    <script src="http://code.jquery.com/jquery-1.5.min.js"></script>
    <script src="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.js"></script>
